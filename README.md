# 🚗 Traction — Car Dealership Inventory System

A full-stack car dealership inventory management system built with **Spring Boot** and **React**, following **Test-Driven Development (TDD)** practices.

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| **Backend** | Spring Boot 3.3, Java 17 |
| **Database** | PostgreSQL 16 |
| **Authentication** | Spring Security + JWT |
| **Frontend** | React 18, TypeScript, Vite |
| **Styling** | Tailwind CSS v3 |
| **Testing (Backend)** | JUnit 5, Mockito, Testcontainers |
| **Testing (Frontend)** | Vitest, React Testing Library |
| **Containerization** | Docker, Docker Compose |
